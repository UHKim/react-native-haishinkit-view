# react-native-haishinkit-view

live view using haishinkit

## Installation

```sh
npm install react-native-haishinkit-view
```

## Usage

```js
import HaishinkitView from "react-native-haishinkit-view";

// ...

const result = await HaishinkitView.multiply(3, 7);
```
